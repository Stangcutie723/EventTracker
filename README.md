EventTracker
============
<br>
This web-app lets the user add new events, edit them, delete them, and type in a date for the event.
<br>
The ruby version I used was 4.0.0.
The Dependencies are as follows:<br>
  coffee-rails (~> 4.0.0)<br>
  jbuilder (~> 1.2)<br>
  jquery-rails<br>
  rails (= 4.0.0)<br>
  sass-rails (~> 4.0.0)<br>
  sdoc<br>
  sqlite3<br>
  turbolinks<br>
  uglifier (>= 1.3.0)<br>
